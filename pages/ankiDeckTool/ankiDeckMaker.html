Anki Deck Creator

How it works:
• Press Enter inside Front or Back to move to a new row.
• If you only fill Front and press Enter, new Front rows are created automatically.
• Later, paste return-separated Back values into any Back cell and they will distribute automatically.
• Export creates a tab-separated .txt file for importing into Anki.
# Front Back
Export format: Front[TAB]Back
Compatible with import into TSV / tab-separated Anki decks.
